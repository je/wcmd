Wilderness Character Database

This is the front page of an application for tracking wilderness character trends in NWPS wilderness areas.

  • Things we have:
    • user accounts
    • measure selection and trends
    • import/export
    • basic aggregation
    • tags/group filters
    • and everything you see here

  • Things we don't:
    • custom roles/permissions
    • custom data entry forms
    • values to trend (with protocols)
    • locally developed measures
    • trend data graphics
